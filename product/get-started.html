Get started with Snowplow today

There are two places you can go to get started with Snowplow today:

  • Get in touch with the Snowplow team. We can setup and host Snowplow for you
  • Download the source code and get started yourself. Snowplow is open source. Review our setup guide to learn what is involved in getting Snowplow up and running yourself

A step-by-step guide to setting up Snowplow can be found on our wiki.

Help and support

For help and support getting up and running with Snowplow, see our professional services details, or get in touch.
